# Getting Started with Create React App

This project uses inversify as dependency injection library and contains multiple examples of of how to use inversify.js with [react](https://github.com/facebook/create-react-app) and [mobx](https://github.com/mobxjs/mobx)

### Here are all DI modules

[`inversify`](https://github.com/Kukkimonsuta/inversify-react)
[`inversify-react`](https://github.com/inversify/InversifyJS)
[`reflect-metadata`](https://github.com/rbuckton/reflect-metadata)

### Here are all react modules

[`mobx`](https://github.com/mobxjs/mobx)
[`mobx-react`](https://github.com/mobxjs/mobx/tree/main/packages/mobx-react)
